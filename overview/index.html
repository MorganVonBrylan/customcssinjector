What's CSS? How do I learn it?

CSS is the language web developers use to tell the browser how to style a page (think text color, spacing, background images, etc). You can learn it on W3Schools. The Mozilla Developer Network also has great resources, although it's mostly documentation for people who already know some CSS rather than tutorials.

Oh and you can open your browser's developer tools with F12 or Ctrl+Shit+I. They're very handy!

Global rules, domain rules, URL rules

Global rules apply on all websites.

Domain rules apply on the website you are currently in (for instance if you're on Youtube, it will apply on all www.youtube.com pages).

Finally, URL rules apply on the specific page you are on. Note that search parameters (the ones after ?) are ignored. For instance if you edit URL rules while on https://www.youtube.com/watch?v=jNQXAC9IVRw, those rules will apply on every www.youtube.com/watch page no matter what that ?v= is.

Rules are applied in that order. That means that with an identical specificity, URL rules will overwrite domain rules, which overwrite global rules.

Whitelist and blacklist

If you have a whitelist, only websites in that whitelist will get custom CSS, even if you have domain or URL rules set.

Domains in the blacklist will never get custom CSS, even if they are also in the whitelist.

Currently, Crayon considers the full domain: for example, whitelisting www.google.com will not include mail.google.com. You can however use a wildcard for the subdomain (and currently, only for the subdomain): *.google.com will match mail.google.com, www.google.com, google.com, etc.

Semi-advanced use: whitelist all subdomains of a website except some by whitelisting its wildcarded domain and blacklisting specific subdomains.
